## 安装

在项目根目录的 settings.gradle 添加仓库

```kotlin
dependencyResolutionManagement {
    repositories {
        // ...
        maven { url 'https://jitpack.io' }
    }
}
```

然后在 module 的 build.gradle 添加依赖

```groovy
implementation 'com.github.liangjingkanji:Tooltip:1.2.3'
```
